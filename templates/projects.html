Colorimetry App

Swift, Firebase

An imaging processing app developed to output and manipulate RGB values from photos. Intended for quantifying chemical levels in colorimetric reactions. The app features a data log that stores measurements along with temperature, humidity, and gps coordinates. The log also interfaces with an online data center through Firebase.

Shark Classifier

Python

Convolutional neural network built to classify different species of sharks. Images were collected through the use of the Flickr API. Many difficulties to be remedied due to lack of data, unbalanced classes, and quality of data. Work in progress.

RGB Data Center

Python, Flask, Firebase, HTML, CSS

Online data center built to interface with the Colorimetry app through Firebase. The center stores colorimetric measurements made by different users. The data center also utilizes the Tom-Tom Map API to set pins and relevant information of each measurement on a world map.

Infrared Box

Python, Raspberry Pi

Infrared imaging box built with as Raspberry Pi and NoIR infrared camera. The box was built to take advantage of colorimetric reactions that have absorption peaks in the infrared. In particular four LEDs at 850nm were used to image the molybdenum blue reaction and improve phosphate qunatification sensitivity.
